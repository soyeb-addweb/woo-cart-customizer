Simple Customization of Add to Cart Button is a flexible plugin that allows you to easily change "cart" names, messages, notices, and button text from the WooCommerce admin panel.

== Description ==

Simple Customization of Add to Cart Button is a feature-rich plugin designed to simplify the customization of "cart" and button text in WooCommerce. Often, such changes require extensive edits to WooCommerce core files, which can be tedious and error-prone. 

With this plugin, you can modify the "cart" name, messages, notices, and button text for different product types (Simple, Grouped, External, Variable) from a centralized WooCommerce settings panel. Save time and effort while keeping your WooCommerce store professional and tailored to your needs.

**Key Features:**
- Customize "Add to Cart" button text for all product types (Simple, Grouped, External, Variable).
- Change "cart" labels and related messages easily.
- Admin panel integration for a seamless user experience.
- Compatible with the latest versions of WordPress and WooCommerce.

== Installation ==

**Installation Instructions:**

1. Download the zip file and extract its contents.  
2. Upload the `custom-cart-button` folder to your `wp-content/plugins/` directory.  
3. Activate the plugin through the "Plugins" menu in WordPress.  
4. Navigate to WooCommerce -> Settings -> Integration Tab to configure settings.
